# Чек-лист ручного тестирования

Адрес для проверки: `http://localhost:8000/?balance=30000&reserved=20001`

## Главная страница

- [x] Отображаются три счёта: Рубли, Доллары, Евро
- [x] Баланс рублёвого счёта берётся из адресной строки
- [x] Доллары: 100 $, евро: 300 €

## Перевод с рублёвого счёта

- [x] По клику на «Рубли» открывается форма «Перевод на карту»
- [x] В поле карты принимаются только цифры
- [x] Номер форматируется группами по 4 цифры
- [x] Форма суммы появляется при 16 цифрах в номере карты
- [ ] **При 17 цифрах форма не должна открываться** — дефект D-01
- [x] Положительные суммы вводятся
- [ ] **Минус в сумме должен отбрасываться** — дефект D-02
- [x] При сумме 1000 комиссия равна 100
- [ ] **При сумме 99 комиссия должна быть 9** — дефект D-04
- [x] При превышении лимита — «Недостаточно средств»
- [ ] **При равенстве лимиту перевод должен быть разрешён** — дефект D-03
- [x] После «Перевести» появляется alert

## Замечания

- [x] У блока комиссии опечатка в id: `comission` вместо `commission`
- [x] В консоли браузера остаются отладочные сообщения
